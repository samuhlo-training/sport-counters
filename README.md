### 01 __ ARCHITECTURE & DECISIONS

| COMPONENT | TECH | NOTE |
| :--- | :--- | :--- |
| **Runtime** | `Bun` | Fast all-in-one JavaScript runtime |
| **Server** | `Express` | Minimalist web framework for Node/Bun |
| **Database** | `Neon DB` | Serverless PostgreSQL with branching |
| **ORM** | `Drizzle` | TypeScript ORM for peak performance and type safety |

<br>

### 02 __ INSTALLATION

*Run local environment:*

```bash
# 1. Clone
git clone https://github.com/samuhlo-training/sport-counters.git

# 2. Install dependencies
bun install

# 3. Ignite
bun run dev
```
